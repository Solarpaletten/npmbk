Чтобы использовать этот скрипт:

1. Создайте новую директорию для проекта:
```bash
mkdir npmbk && cd npmbk
```

2. Создайте файл setup-project.sh и вставьте в него код выше:
```bash
touch setup-project.sh
# Вставьте код выше в setup-project.sh
```

3. Сделайте скрипт исполняемым и запустите его:
```bash
chmod +x setup-project.sh
./setup-project.sh
```

После выполнения у вас будет полностью настроенный проект с:
- Структурой папок
- Всеми необходимыми файлами
- Установленными зависимостями
- Готовым скриптом создания админа

Для запуска:
```bash
# Создание админа
node scripts/user-management.js

# Запуск сервера
npm run dev
```

Теперь вы можете:
1. Войти используя email: solar@solar.de и пароль: user123
2. Использовать полученный токен для доступа к защищенным маршрутам
3. Работать с админ-панелью
